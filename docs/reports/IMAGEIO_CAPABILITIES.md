## Evidence

Generated by:

```bash

```

The app reads `CGImageSourceCopyTypeIdentifiers()` and `CGImageDestinationCopyTypeIdentifiers()` at runtime rather than assuming that every OS version exposes the same codecs. WebP is the exception: decoding uses ImageIO while encoding uses a statically linked libwebp 1.6.0 bridge built for arm64/macOS 14.
